Задача:
1) Прочитай страницу слева направо, сверху вниз.
2) Верни JSON по схеме. Ничего не выдумывай: только то, что реально прочитано как текст.
   Это OCR-задача: запрещено описывать изображения, иконки и графику. Не переводи текст.
3) raw_lines: все строки по порядку; если неразборчиво — "???".
4) blocks: иерархия блоков. Используй вложенность:
   - создавай блоки типа section при наличии заголовка и вкладывай в него связанные блоки;
   - для списков: list -> list_item;
   - для таблиц: table -> table_row (в children) при возможности.
   Каждый блок обязан содержать:
   - type: тип блока (title, heading, section, paragraph, list, list_item, table, table_row, caption, header, footer, other).
     Тип figure используй только если на странице есть явная подпись/метка к изображению.
   - text: основной текст блока.
   - level: целое число (1 для верхнего уровня, 2 для дочерних и т.д.).
   - lines: строки блока в порядке чтения (подмножество raw_lines), минимум 1 строка.
   - table: {headers: [...], rows: [[...]]} для таблиц, иначе пустые массивы.
   - meta: массив {key, value} для доп. атрибутов, иначе [].
   - children: массив дочерних блоков, иначе [].
5) extra: массив {key, value} для общих заметок страницы, иначе [].
6) Если на странице нет читаемого текста, raw_lines = [], blocks = [].
7) Ответ строго JSON по схеме, без лишнего текста.
