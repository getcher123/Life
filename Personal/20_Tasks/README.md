# Tasks (Personal)

Задачи хранятся как чекбоксы `- [ ] ...` (плагин Tasks).

Рекомендуемый шаблон: `00_System/Templates/Task - Personal.md`.

Если задача относится к проекту/цели — добавляй в строку задачи ссылку на файл (`[[PP-...]]`, `[[PG-...]]`), чтобы работали запросы на страницах проекта/цели.
- Пиши **человеческий заголовок** в `# ...`, а ID файла оставляй только для навигации.
- Не оставляй пустые свойства (`area`, `goal`, `project`) — добавляй только релевантные.
- Для больших задач используй одну задачу + `## Checklist` внутри (а не много мелких файлов).
- Ссылки в строке задачи лучше давать с алиасами: `[[PP-...|Название проекта]]`.
