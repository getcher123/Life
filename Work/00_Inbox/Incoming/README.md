# Incoming materials (Work)

Сюда складываются входящие **не‑md** файлы (DOCX/PPTX/PDF/аудио/XLSX/CSV/изображения) для разбора агентом.
Канонический формат в vault — `.md`; не‑md файлы считаются источниками/артефактами.

Протокол:
1) ты кладёшь файлы в `Work/00_Inbox/Incoming/` (они не коммитятся в Git);
2) агент конвертирует файлы в заметки `Work/50_Notes/` (с `## Summary`);
3) агент задаёт уточняющие вопросы и **только после подтверждения** создаёт задачи/проекты;
4) исходники перемещаются в `Work/99_Archive/Incoming/` (вместо удаления).

Экспорт наружу (DOCX/PDF) делается из `.md` и хранится в `Work/50_Notes/Exports/` или в `Work/90_Assets/` для проектных артефактов.

Команда агента:
- `scripts/process-incoming.sh --domain work`
